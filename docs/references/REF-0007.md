MCP is the standardized seam between agents and this store. The server exposes
the memory verbs and substrate intents as MCP tools; every client (Claude Code,
inspector, remote bridges) reaches the same transport-free core through it.

## Our reading of the spec

- The spec standardizes the wire; it does not choose your vocabulary. ADR 0106
  chooses intent verbs over generic CRUD at this boundary.
- Dated revisions (2025-11-25 current at verification time) with explicit
  version negotiation make the SDK dependency a stable, boring choice.
- Tool descriptions are the discovery surface for Tool Search — written for the
  agent deciding whether to load the tool, not for API docs.
