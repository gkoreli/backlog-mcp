title: '@parcel/watcher — native filesystem watcher'

The docs folder is the source of truth, and the tool is one of several writers
— humans with editors and git are the others. @parcel/watcher is how the server
notices the others. It provides coalesced, native filesystem events across
macOS/Linux/Windows, which the docs-tree watcher translates into catalog and
index updates and SSE pushes to the viewer.

Pinned exact (2.5.6, no caret) because prebuilt native binaries make loose
ranges a supply-chain and rebuild hazard.
