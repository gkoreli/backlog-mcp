title: gray-matter — frontmatter parser and serializer

Frontmatter markdown is the load-bearing format of the whole product — "a
teammate reads every artifact as plain frontmatter markdown on GitHub" is half
of the Cold-Open Test. gray-matter is the small, boring, correct library that
parses and serializes that format everywhere the storage layer touches disk.

It matters that reads are lenient (unknown fields preserved, missing
frontmatter tolerated) and writes are canonical (ordered, validated) — the
codec supports both without ever reformatting a human-authored file.
