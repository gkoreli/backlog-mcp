claude-mem answers a question we have deliberately left open: where does
memory COME FROM when nobody writes it deliberately? Its answer — observe the
agent's tool calls, compress trajectories into observations — is the
strongest capture mechanism in the claims space, and it composes with our
store better than with its own: the observations could be curated claims in
markdown instead of rows in a hidden SQLite file.

## Cautionary half

The same project demonstrates the failure modes we design against: memory you
cannot open in an editor, and a localhost daemon that shipped unauthenticated.
Files-only, no-daemon is not conservatism — it is the security posture.
