ADR 0113 made document types data: a project declares a substrate as JSON with
a bounded JSON Schema, and the server compiles it into a canonical-write
validator. Ajv is the compilation target — the strict Draft 2020-12 engine that
enforces the declared shape on every write.

## The trust split

A schema from a cloned repo is untrusted input. The meta-schema gate rejects
everything outside a small allowlisted subset before Ajv ever sees it; Ajv then
runs with strict mode on and every mutation feature (coercion, defaults,
removeAdditional) off. Validation power without an execution surface.
