# PROMPT 0006 — verbatim

> For the second [the memory flywheel] we have to be careful, yes i want you to
> write memory artifacts but we shouldnt summarize and compress information, and
> memory artifacts shouldnt be written after the fact, like ine agent who is
> doing the wirk it should write memory srtifact at some point maybe after the
> work is finished ir at some checkpoint. If another agent assesses the work and
> decides to summarize and write the memory we are going to have a broken
> compressed telephone and memories quality will degrade a lot. So we have to be
> mindful who writes the memory when and on what trigger.

Distilled as the **first-person capture law** in the memory-flywheel proposal:
the agent that did the work writes the memory, at completion or at its own
checkpoints, in its own words — never a post-hoc summarizer. Assessors may
record their own first-person assessments as their own memories; they never
ghost-write the doer's.
